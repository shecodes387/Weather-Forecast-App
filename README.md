# 🌤️ Weather Forecast Application

A Java Swing-based desktop application that provides real-time weather updates using the OpenWeatherMap API.

## Features

- Search weather by city
- Real-time temperature
- Weather conditions
- Humidity
- Wind speed
- Weather icons
- User-friendly GUI

## Technologies

- Java
- Java Swing
- OpenWeatherMap API
- JSON

## How to Run

1. Open the project in NetBeans or any Java IDE.
2. Add the required JSON library.
3. Run `WeatherApp3.java`.
4. Enter a city name and click **Get Weather**.
